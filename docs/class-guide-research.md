# WoW Classic class guide research notes

The in-app Classic class guides are condensed, original summaries built from common guide structures used by established WoW Classic references:

- Wowhead's Classic class hub organizes class content around leveling specs, talent builds, rotations, best-in-slot gear, macros and addons.
- Wowhead's Classic leveling overview highlights class-specific leveling guides, spell purchase choices, leveling talents, gear progression and profession support.
- Icy Veins' Classic leveling guide emphasizes that class-specific guides should supplement general 1-60 routing and that realistic player pacing differs from speedrun records.
- Icy Veins class pages commonly split guides into introduction, talents, rotation/cooldowns, stat priority, enchants/consumables, gear and leveling.

These notes intentionally avoid copying external guide text. The app data summarizes stable Vanilla/Classic concepts into compact cards for product prototyping. Production content should add citations per guide, patch/version metadata and reviewer notes from class specialists.
